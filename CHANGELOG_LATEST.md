- option for 'heatingfailsafe': will cause boiler to turn off after 120 seconds (default) if no EMS messages are received from HA, etc

- added SK (Slovenian) language. Thanks @misa1515
- CPU info [#1497](https://github.com/emsesp/EMS-ESP32/pull/1497)

- boiler Bosch C1200W, id 12, [#1536](https://github.com/emsesp/EMS-ESP32/issues/1536)
